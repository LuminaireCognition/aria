# Ore Database

Complete reference for mining operations. Organized by security status accessibility.

## Minerals Overview

| Mineral | Primary Source | Used For |
|---------|----------------|----------|
| **Tritanium** | Veldspar, Scordite | Everything (base mineral) |
| **Pyerite** | Scordite, Pyroxeres | Most T1 items |
| **Mexallon** | Plagioclase, Pyroxeres | Mid-tier manufacturing |
| **Isogen** | Omber, Kernite | Advanced T1, some T2 |
| **Nocxium** | Jaspet, Hemorphite | Advanced manufacturing |
| **Zydrine** | Low/nullsec ores | T2 components |
| **Megacyte** | Nullsec ores only | T2 components, capitals |
| **Morphite** | Mercoxit only | T2 manufacturing |

## High Security Ores (1.0 to 0.5)

| Ore | Volume | Minerals | Notes |
|-----|--------|----------|-------|
| **Veldspar** | 0.1 m³ | Tritanium | Most abundant; best trit source |
| **Scordite** | 0.15 m³ | Tritanium, Pyerite | Good starter ore |
| **Pyroxeres** | 0.3 m³ | Pyerite, Mexallon | Also in lowsec |
| **Plagioclase** | 0.35 m³ | Tritanium, Mexallon | Also in lowsec |

**Highsec mining focus:** Veldspar for Tritanium, Scordite for Pyerite.

## Ore Variants

All standard ores have improved variants with better refining yields:

| Grade | Yield Bonus | Availability |
|-------|-------------|--------------|
| Base | 0% | All locations |
| II-Grade | +5% | Less common |
| III-Grade | +10% | Rare |
| IV-Grade | +15% | Very rare |

**Prioritize higher grades when available** - same mining time, more minerals.

## Venture Optimization

Your Venture has 5,000 m³ ore hold. Optimal loads:

| Ore | Units per Full Hold | Time to Fill (approx) |
|-----|---------------------|----------------------|
| Veldspar | 50,000 | Long (small volume) |
| Scordite | 33,333 | Medium |
| Pyroxeres | 16,666 | Medium |
| Plagioclase | 14,285 | Medium-fast |

**Tip:** Higher volume ores fill hold faster but yield less per m³. Balance based on what minerals you need.

## Self-Sufficient Mining Priority

For T1 manufacturing, prioritize in this order:
1. **Tritanium** - Veldspar (always needed)
2. **Pyerite** - Scordite
3. **Mexallon** - Plagioclase or Pyroxeres
4. **Isogen** - Requires lowsec (Omber, Kernite)
5. **Nocxium** - Requires lowsec (Jaspet, Hemorphite)

**Note:** Zydrine and Megacyte require null/wormhole access or reprocessing loot drops.

## Compression

Compressed ore:
- 100:1 volume reduction
- Same mineral yield when reprocessed
- Requires Upwell structure with Standup Reprocessing Facility

For hauling ore to better reprocessing facilities, compression is essential.
