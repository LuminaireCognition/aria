# Gallente Ship Progression

Training roadmap for Gallente pilots. Includes mining (ORE) and exploration paths.

## Gallente Combat Philosophy

- **Primary Tank:** Armor
- **Primary Weapons:** Hybrid turrets (blasters/railguns), Drones
- **EWAR Specialty:** Sensor dampeners
- **Damage Types:** Kinetic/Thermal

## Combat Ship Progression

### Frigates → Destroyers → Cruisers

```
Tristan (Drone Frig) ──────┐
                           ├──→ Algos (Drone Dessie) ──→ Vexor (Drone Cruiser)
Incursus (Blaster Frig) ───┘                              │
                                                          ├──→ Vexor Navy Issue
                                                          └──→ Ishtar (T2 HAC)
```

## Mining Ship Progression

Mining ships are ORE faction (cross-faction, no empire skills needed).

```
Venture ──→ Mining Barges ──→ Exhumers
              │                  │
              ├─ Procurer ──→ Skiff (Tank)
              ├─ Retriever ──→ Mackinaw (Capacity)
              └─ Covetor ───→ Hulk (Yield)
```

> **Reference:** See `reference/mechanics/mining_ships.md` for complete ship specifications and comparison.

### Venture Stats

- **Ore Hold:** 5,000 m³
- **Mining Bonus:** +5% yield/level (Mining Frigate)
- **Special:** +2 warp core strength, gas harvesting bonus
- **Skill Required:** Mining Frigate I

### Recommended Path (Self-Sufficient)

1. **Venture** - Free from career agent
2. **Procurer** - Best tank for solo in any space
3. **Skiff** - T2 Procurer when ready for exhumers

## Exploration Ship Progression

```
Imicus (T1) ──→ Helios (Covert Ops)
                    │
                    └──→ Astero (Faction, optional)
```

### Exploration Frigates

| Ship | Class | Virus Strength | Scan Bonus | Special |
|------|-------|----------------|------------|---------|
| **Imicus** | T1 Frigate | +5 | +7.5%/level | 400m³ cargo |
| **Helios** | Covert Ops | +10 | +10%/level | Covert cloak |
| **Astero** | Faction | +10 | +37.5% (fixed) | Combat capable |

### Imicus Stats

- **Scan Bonus:** +7.5% probe strength/level
- **Virus Bonus:** +5 strength (role)
- **Slots:** 3H / 4M / 3L
- **Skill Required:** Gallente Frigate I

### Helios Stats

- **Scan Bonus:** +10% probe strength/level
- **Virus Bonus:** +10 strength (role)
- **Slots:** 3H / 5M / 3L
- **Special:** Covert Ops Cloak, warp while cloaked
- **Skill Required:** Covert Ops I (requires Gallente Frigate V)
