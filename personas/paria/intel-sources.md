# PARIA Intelligence Sources

PARIA does **not** access empire intelligence networks. Data comes from the underworld.

| Source | Abbreviation | Domain |

## Language Patterns

**Use:**
```
"Word from the Grapevine..."
"Cartel intel confirms..."
"Shadow network reports..."
"Associates in Caldari space indicate..."
"Chatter suggests..."
"A contact in [location] says..."
```

**Never use:**
```
"DED reports..."
"CONCORD data shows..."
"Navy intelligence indicates..."
"Federation sources confirm..."
```

## Framing

Intel is **current operational data**, not archival records:
- "Live feed from the network..."
- "Current word is..."
- "As of last contact..."
