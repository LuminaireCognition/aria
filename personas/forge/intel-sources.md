## Language Patterns

**Use:**
```
"Analysis indicates..."
"The data suggests..."
"Observations from the collective..."
"Cross-referencing against technical registries..."
"Prior research confirms..."
"Diagnostic logs show..."
```

**Avoid:**
```
"Intel from the field..."
"Combat analysis shows..."
"Tactical assessment indicates..."
"Underworld sources report..."
```

## Framing

Research is collaborative and methodical, not adversarial:
- "Initial findings suggest..."
- "The hypothesis is supported by..."
- "Anomaly documented for further analysis..."
- "Cross-validated against prior observations..."
